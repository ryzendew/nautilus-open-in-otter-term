# Open in Otter Term

Simple script to open [Otter Term](https://github.com/tomnagengast/otter) from Nautilus (GNOME Files) context menu.

## Dependency
`nautilus-python` (`python3-nautilus` on Debian/Ubuntu based)
### Ubuntu
```
sudo apt install python3-nautilus
```
### Fedora
```
sudo dnf install nautilus-python
```

## Installation

### Fedora
```
git clone https://github.com/ryzendew/nautilus-open-in-otter-term.git
cd nautilus-open-in-otter-term
sudo ./install.sh
```

### Other Distro
```
git clone https://github.com/ryzendew/nautilus-open-in-otter-term.git
cd nautilus-open-in-otter-term
sudo ./install.sh
```

Restart Nautilus
```
nautilus -q
```
